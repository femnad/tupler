tupler
======

A command line Zulip client.

Installation
------------

    python setup.py install

Running
-------

    tupler [--rcfile <tulpre-rcfile>]

Where `rcfile` is:

    {
      "server": "<zulip-server>",
      "email": "<email-of-the-zulip-user>",
      "api_key": "<api-key-of-the-user>",
    }
